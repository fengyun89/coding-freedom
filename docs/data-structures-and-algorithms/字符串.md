# 字符串

字符串的逻辑结构和线性表相似，只是字符串的数据元素都是字符，但字符串的操作和线性表差别很大。

## 模式匹配

### 朴素模式匹配算法

又称为暴力模式匹配

```js
const nativePatternMatch = (text, pattern) => {
    begin: for (let i = 0; i < text.length; i++) {
        for (let j = 0; j < pattern.length; j++) {
            if (text[i + j] !== pattern[j]) {
                continue begin;
            }
        }
        return i;
    }
    return -1;
}
```

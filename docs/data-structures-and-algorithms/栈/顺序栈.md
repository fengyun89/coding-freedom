# 顺序栈

栈的顺序存储实现。

```js
class SequentialStack {
  #data = [];

  // 入栈
  push(element) {
    this.#data.push(element);
  }

  // 出栈
  pop() {
    return this.#data.pop();
  }

  // 查看栈顶元素
  peek() {
    return this.#data[this.#data.length - 1];
  }

  // 判断栈是否为空
  isEmpty() {
    return this.size() === 0;
  }

  // 清空栈
  clear() {
    this.#data = [];
  }

  // 获取栈中元素的个数
  size() {
    return this.#data.length;
  }

  // 打印栈中的元素
  print() {
    console.log(this.#data.toString());
  }
}
```

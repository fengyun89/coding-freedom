# 线性表(List)

数据元素的有序序列。

除了第一个元素，每个元素有且只有一个前驱；除了最后一个元素，每个元素有且只有一个后继。

```
ADT List {
    数据对象：
        D = { ai | ai ∈ ElemSet, i = 1, 2, ..., n, n >= 0 }
    数据关系：
        R1 = { <ai-1, ai> | ai-1, ai ∈ D, i = 2, 3, ..., n }
    基本操作：
        InitList(*L)
        DestroyList(*L)
        ClearList(*L)
        ListEmpty(L)
        ListLength(L)
        GetElem(L, i, *e)
        LocateElem(L, e)
        PriorElem(L, cur_e, *pre_e)
        NextElem(L, cur_e, *next_e)
        ListInsert(*L, i, e)
        ListDelete(*L, i, *e)
        ListTraverse(L, visit())
}
```
